Configuration

Captcha

Open Settings in the admin sidebar, switch to the Captcha tab panel, and fill the Cloudflare Turnstile values there.

Fill these fields

Captcha fields
Captcha usage
Captcha site key
Captcha secret key

Get the keys from Cloudflare Turnstile

  1. Open Cloudflare Turnstile.
  2. Sign in to the Cloudflare Dashboard.
  3. Select Turnstile from the sidebar.
  4. Click Add Site to create a widget.
  5. Provide a site name and add your domain or hostname.
  6. Choose the widget mode: Managed, Non-interactive, or Invisible.
  7. Click Create to generate the site key and secret key.

Save it in Quackbase

  1. Go back to Settings -> Captcha.
  2. Set Captcha usage.
  3. Paste the Captcha site key.
  4. Paste the Captcha secret key.
  5. Click Save captcha settings.

Widget modes

Managed

Cloudflare determines the interaction level for the visitor.

Non-interactive

No user interaction, just a passive check.

Invisible

Runs in the background while still protecting the form.

Recommended check

Submit a real form once after saving to confirm widget loading and server-side validation both work.

Validation flow

High-level flow
Visitor opens contact form
-> completes captcha challenge
-> form submits to /api/forms/contact
-> server verifies captcha token
-> valid submissions continue to save / notify
