FieldDescriptionRequiredExample / Notes

Cloudflare Turnstile setup

Open Cloudflare Turnstile

  1. 1

    Sign in to the Cloudflare Dashboard.

  2. 2

    Open Turnstile from the sidebar.

  3. 3

    Click Add Site.

  4. 4

    Enter a site name.

  5. 5

    Add your domain hostname.

  6. 6

    Choose a widget mode.

  7. 7

    Click Create.

  8. 8

    Copy the Site Key and Secret Key.

  9. 9

    Return to the admin dashboard and paste them into Settings → Captcha.

  10. 10

    Save the captcha settings.

Managed

Cloudflare automatically decides the required interaction level.

Non-interactive

Performs a passive check without direct user interaction.

Invisible

Runs in the background and only challenges when needed.

Site Key

Used in frontend HTML and widget rendering.

Secret Key

Used on the server to verify captcha responses.

Secret handling

Never expose the secret key in frontend code.
