Getting Started

Quickstart

Deploy first, then finish settings in the admin dashboard.

Fastest path

  1. 1

    Deploy the project

    Use the repository deployment flow and provision the Cloudflare resources required by the app.

  2. 2

    Wait for the first release

    Confirm that the Worker deployment and initial database setup complete successfully.

  3. 3

    Open the admin dashboard

    Visit /admin on the deployed site and create the initial administrator account.

  4. 4

    Configure runtime settings

    Set up media storage, captcha, and email only when your site actually needs those services.

After first login

Review branding, navigation, and publishing defaults before adding production content.
