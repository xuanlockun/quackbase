Architecture

Cloudflare Worker, D1, R2, cache, and delivery in one view.

Runtime

Cloudflare Workers run app and admin requests at the edge.

Database

D1 stores content, settings, sessions, forms, and access data.

Object storage

R2 or S3-compatible storage handles uploaded media.

Cache

Edge cache reduces repeated requests for public content.

Delivery

Static assets and responses can be served close to users.

Security

Admin APIs stay behind sessions and permission checks.
