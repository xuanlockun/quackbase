Media Settings

Step-by-step configuration

  1. 1

    Open the admin dashboard

    Sign in with an account that can manage site settings.

  2. 2

    Open Settings

    Go to Settings from the admin sidebar.

  3. 3

    Select Media

    Choose the Media tab panel.

  4. 4

    Fill all storage fields

    Enter the endpoint, bucket, public base URL, region, path-style option, access key ID, and secret access key.

  5. 5

    Test the connection

    Click Test Connection to validate the credentials and bucket configuration.

  6. 6

    Save the settings

    If the connection succeeds, click Save Media Settings.

Successful validation

If the test connection succeeds, the storage credentials and bucket configuration are valid.

Credential safety

Keep access keys private. Do not commit them to GitHub or expose them in frontend code.
