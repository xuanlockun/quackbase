Email Settings

Configure email delivery with Resend.

Current provider

Resend is the current provider. Create an account and generate an API key.

Admin path

UI Path

Settings Admin Sidebar → Email Tab Panel

Provider

Resend

Website: https://resend.com
Required credential: Resend API Key

Setup steps

  1. 1

    Create or sign in to a Resend account.

  2. 2

    Verify your sending domain if required.

  3. 3

    Open API Keys in the Resend dashboard.

  4. 4

    Create a new API key.

  5. 5

    Copy the API key.

  6. 6

    Paste it into the Email settings in the admin dashboard.

  7. 7

    Save settings.

  8. 8

    Send a test email if the app provides a test action.

Field reference

FieldDescriptionRequiredExample / Notes
Resend API keyAPI key used to send transactional emails through Resend.YesKeep private and rotate if leaked.

Credential safety

Do not expose the Resend API key in client-side code. Store it as a secure environment variable or encrypted setting.
