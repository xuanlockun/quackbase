Email

Quackbase currently uses Resend for email delivery. Open Settings in the admin sidebar, switch to the Email tab panel, and save your sender details plus the Resend API key.

What you need

  • A Resend account
  • A Resend API key
  • A sender email address
  • An optional sender name

Recommended flow

  1. Create an account at resend.com.
  2. Generate a Resend API key from the Resend dashboard.
  3. In Quackbase, open Settings -> Email.
  4. Fill Sender email and Sender name.
  5. Paste the Resend API key.
  6. Click Test email.
  7. If the test succeeds, click Save Resend settings.

Writing help

The shortest version is: register on Resend, create an API key, paste it into Quackbase, test it, then save. For production, verifying your sending domain in Resend is recommended so mail is less likely to land in spam.
